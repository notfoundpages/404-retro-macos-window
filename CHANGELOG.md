## [1.0.0] - 2026-03-19

### Added

- Retro macOS-inspired 404 page with authentic desktop-style layout
- Interactive title bar control buttons styled after macOS window controls
- Dynamic background color switching triggered by title bar controls
- CSS grid background pattern with parallax-style visual depth
- Responsive layout for mobile, tablet, and desktop screens
- Multiple preview assets for desktop, mobile, and social sharing
- Multiple color variants including blue, red, yellow, and green
- Semantic HTML structure for improved accessibility
- Keyboard-accessible interactive controls
- Project documentation including README, contribution guide, and code of conduct
- MIT license
